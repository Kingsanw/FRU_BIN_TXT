        This tool allows you to parse FRU binary files and view their contents in a human-readable format.
        ------------------------------------------------------------------------ 
        Features:
            1. Select FRU BIN File: Load a FRU binary file for parsing.
            2. Save Parsed Result: Save the parsed output to a text file.
            3. BIN_VS_Txt: Compare the parsed FRU binary output with an existing FRU text file.
            4. Text_Vs_Text: Compare two FRU text files for differences.
            5. BIN_Vs_BIN: Compare two FRU binary files for differences.
            6. Help: Display this help information.
            7. Quit: Exit the application.
        Usage:
             - Click 'Select FRU BIN File' to choose a binary file.
             - After parsing, you can save the result or compare it with other files.
        ------------------------------------------------------------------------ 
        For more information, please refer to the documentation or contact support.
